## Architecture Decisions
- CLI-first（Click），不引入 web framework
- 設定 / sessions / user SOPs 都在 `~/.adjutant/`
- SOP 用 markdown + YAML frontmatter，和 CrossVal persona 相同模式
- 檔案操作限制在 notebook_root 內，防止路徑穿越
